# 系统配置
## 大数据服务器以及网络推荐配置
大数据服务器一般需要三台以上主机作为主节点，以及数台（具体数量按照实际数据量增减）数据节点。为保证高效IO，应为不同的服务（数据量较大的服务）配置不同的硬盘。推荐配置如下：
操作系统兼容性：大数据服务器的操作系统对Linux内核版本2.6以上版本全面支持。例如发行版CentOS6.5、SUSE11SP2以上版本。
